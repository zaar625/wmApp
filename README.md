# Work & Manage app 
매장의 생산성을 향상시키기 위한 알바생 출/퇴근 관리 및 공지사항 공유앱

### 프로젝트 목표
1. 실세 매장에서 사용할 수 있도록 서비스 배포(구글콘솔)
2. 클린 코드강의를 바탕으로 컴포넌트 별 의식적인 리팩토링 연습
3. 실 유저에게 개선사항 피드백 받기
